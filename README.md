# Лабораторная работа №14 "Работа с библиотекой Pillow"

## Цели
1. Ознакомиться с библиотекой Pillow для работы с изображениями
2. Освоить ООП подход при работе с изображениями
3. Развить навыки обработки изображений

## Задачи
### Задание 1
Создать два связанных класса:
- ImageHandler - базовая работа с изображениями
- ImageProcessor - обработка и применение эффектов

### Задание 2 (Вариант 5)
1. В ImageHandler:
   - Масштабирование до 50%
   - Сохранение с добавлением текущей даты
2. В ImageProcessor:
   - Применение фильтра "Эмбосс"
   - Добавление водяного знака "Вариант 5"

## Структура проекта
- task_1/ - общие классы
- task_2/ - индивидуальное задание (вариант 5)

## Требования
- Python 3.6+
- Библиотека Pillow
